## 2. 核心功能

1. **长效种子（Long-Seeding）**\
   独有“长效做种”算法，冷门资源也能保持 30+ 可用源，大幅提升存活率与速度[^5^]。
2. **全协议下载**\
   同一界面搞定 BT、Magnet、HTTP、FTP、eD2k，甚至可挂载 eMule 插件。
3. **NAT1 UDP 打洞**\
   全球首款支持 NAT1 Full-Cone 打洞的 BT 客户端，内网用户也能拿到 High-ID，上传下载双向提速[^9^]。
4. **磁盘缓存 & 32 MB 区块对齐**\
   降低机械盘随机读写，延长硬盘寿命；大文件一次性落盘，减少碎片[^7^]。
5. **边下边看**\
   视频文件下载到 5% 即可预览，内置解码器，支持倍速播放与字幕挂载。
6. **反吸血 & 自动订阅 Tracker**\
   内置“迅雷吸血”屏蔽列表；每次启动自动拉取热门 Tracker，冷门热门资源都能秒连[^11^]。

## 4. 快速上手（3-Step）

1. **下载**\
   解压后运行 `BitComet_x64.exe`（64 位）或 `BitComet.exe`（32 位）[^2^]。
2. **基础设置**\
   选项 → 任务设置 → 默认下载目录 → 改为非系统盘；\
   选项 → BT 下载 → 勾选“启用长效种子”与“自动订阅 Tracker”。
3. **开始下载**\
   拖拽 `.torrent` / 粘贴磁力 → 选择分文件 → 设定优先级 → 立即下载。

## 5. 高阶玩法

| 场景     | 操作路径                                                |
| ------ | --------------------------------------------------- |
| PT 挂机  | 选项 → 任务 → 禁用 DHT、PEX、LSD；限速上传 80% 带宽[^9^]           |
| Web 远控 | 选项 → 远程下载 → 勾选“启用网页接口”，设端口与 token[^7^]              |
| RSS 订阅 | 侧边栏 → 订阅 → 添加 PT 的 RSS 链接，实现自动抢种                    |
| NAT 检测 | 工具 → NAT 类型检测 → 若显示 Full-Cone，再到“连接”里开启 UDP 打洞[^9^] |
| 边下边看   | 下载列表 → 右键视频 → 预览，可提前 30 min 追剧                      |

## 6. 常见问题速解

| 问题       | 解决方案                                  |
| -------- | ------------------------------------- |
| 下载 0% 不动 | 更新 Tracker 列表 → 工具 → 订阅 → 立即更新        |
| 上传限速失效   | 检查是否开启“反吸血”，部分客户端被误判                  |
| 端口阻塞     | 申请公网 IP → 路由器打开 UPnP 或手动映射 TCP/UDP 端口 |
| 界面英文     | 菜单 → View → Language → 简体中文（绿色版已默认）   |
